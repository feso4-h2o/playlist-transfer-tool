# Fixture Data

This directory contains sample JSON fixtures for tests, local mock transfers,
manual QA, and future replay-style matching work.

The fixtures are metadata-only. They may include public track titles, artists,
albums, durations, release dates, ISRCs, platform track IDs, sample playlist
ordering, search queries, candidate ranks, scores, and matching evidence. They
must not include credentials, cookies, OAuth tokens, request headers, raw
private API payloads, audio, lyrics, cover art, SQLite databases, generated
reports, or local configuration.

The Spotify and QQ Music metadata is included only as sample data for this
project's test and mock workflows. This project is not affiliated with Spotify,
QQ Music, Tencent Music Entertainment, or their rights holders.
